MD-Buildings V2

Import building data from Moldovan public datasets.

Quick Start

  • Select a profile in the sidebar.
  • Ctrl + Double Click on a building to import.
  • Or press Ctrl+Shift+1 with a building selected.

Import Modes

  • Full Import: Geometry and tags.
  • Geometry Update: Only the shape.
  • Tags Update: Only the tags.

For more details, visit the Online Documentation.
